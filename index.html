Jogo no Scratch

Descrição

Scratch é uma plataforma de programação visual desenvolvida pelo MIT Media Lab que permite a crianças e jovens criarem seus próprios projetos interativos, histórias animadas, jogos e muito mais. Através de uma interface de programação fácil de usar, os usuários podem arrastar e soltar blocos de código para criar animações, jogos e outros projetos divertidos, sem precisar escrever código em linguagem de programação convencional. O site do Scratch oferece acesso gratuito à plataforma, bem como recursos de aprendizagem, projetos compartilhados por outros usuários, fóruns de discussão e muito mais. Com a ajuda do Scratch, as crianças podem desenvolver habilidades de pensamento crítico, resolução de problemas, criatividade e colaboração, ao mesmo tempo em que se divertem explorando o mundo da programação.

Explicação sobre Arduíno

Descrição

Arduino é uma plataforma de prototipagem eletrônica de código aberto, que inclui hardware e software para criar dispositivos interativos e projetos eletrônicos. O hardware é baseado em uma placa de microcontrolador, com várias entradas e saídas digitais e analógicas que podem ser programadas usando uma linguagem de programação simplificada baseada em C/C++. A plataforma é muito popular entre os criadores e entusiastas de eletrônica e robótica, pois é fácil de usar, acessível e flexível o suficiente para suportar uma ampla variedade de projetos, desde robôs autônomos e sensores de ambiente até sistemas de automação residencial e controladores de jogos. O ecossistema Arduino também inclui uma grande comunidade de desenvolvedores, documentação detalhada e uma vasta gama de placas e módulos de expansão, tornando-o uma das ferramentas mais versáteis para prototipagem eletrônica

TinkerCad & Arduino

Descrição

Este circuito mostra um irrigador smart, no intuito de facilitar na irrigação mesmo sem a presença de um ser humano. Para esse projeto nós utilizamos um arduino uno que está conectado a um sensor de umidade, um Motor CC e leds vermelho e verde. O dispositivo funciona de forma que quando a umidade do solo for menor que 100, o motor (sendo utilizado para expelir água) e a luz verde serão ativados para que o solo seja molhado. Quando o solo for irrigado o suficiente, de forma que a umidade do solo seja maior que 100, o motor e a luz verde encerrarão suas atividades, então a luz vermelha acenderá para demonstrar que o dispositivo não está mais ligado, até que o ciclo se repita novamente.
